# 📝 Todo App

毎日のタスクを整理できるシンプルなToDoアプリです。
ブラウザだけで動作し、PWAとしてインストールすることもできます。

## 主な機能

### タスク管理
- タスク追加　/ 削除
- タスク編集
- 完了チェック
- 完了タスク一括削除
- 全タスク削除

### タスク情報
- 期限設定（日時）
- 優先度設定（高 / 中 / 低）
- 通知設定（5分前 / 30分前 / 1時間前）

### 通知機能
- 期限前にブラウザ通知
- 通知タイミング選択
- タスク編集時に通知再設定
- 完了 / 削除時に通知キャンセル

### タスク整理
- 検索機能
- フィルター
  - すべて
  - 未完了
  - 完了
- ドラッグ＆ドロップ並び替え

### タスク統計
- 総タスク数
- 完了数
- 完了率
- **今日のタスク表示**
- **今日の完了率**

### UI機能
- ダークモード
- レスポンシブデザイン
- スムーズアニメーション

### PWA機能
- アプリとしてインストール可能
- オフライン対応

## PWAとしてインストール

Chromeの場合：

1. サイトを開く
2. 「アプリをインストール」をクリック
3. デスクトップアプリのように起動できます
