# Todo App

JavaScriptで作成したシンプルなTodoアプリです。

## 機能

・タスク追加
・タスク編集
・タスク削除
・完了チェック
・期限設定
・優先度設定
・タスク検索
・タスクフィルター（すべて/未完了/完了）
・ドラッグ&ドロップ並び替え
・ダークモード
・タスク統計表示
・localStorage保存

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 使い方

1. タスクを入力して追加
2. チェックを入れると完了
3. フィルターで表示切り替え
4. ドラッグで並び替え
